views.py :

1. index - функция ,которая открывает базавую страницу при входе на сайт
2. list_books - функция ,которая отображает все созданные книги на странице.
Все созданные объекты книг читаются из модели Book. 
Извелечение всех объектов происходит в переменную book командой Book.objects.all() 
3. change_status - функция ,которая отображает все созданные книги на странице с ссылками на их статус.
4. change_status_page - функция для изменения статуса книги,которая отображет страницу со всех информацией о книге.
Книга ищется с помощью метода get_object_or_404() в которую помещается название модели и id по корорым будет найдена книга.
form = BookStatusForm(request.POST, instance=book) - при изменении статуса книги на странице отправляется POST запрос , который в дальнейшем 
будет проверен на правильность внесенных данных а позже сохранен в основной модели 
После успешного изменения статуса пользователь будет перенаправлен на страницу list_books
5. delete_book() - функция , которая отображает страницу со всеми книгами которые можно удалить из базы данных 
После отправки POST запроса на страницу get_object_or_404 ищет данную запись по id который бвл отправлен в POST запросе 
При нахождении книги она будет удалена из будет
При не нахождении книги вызывается блок исключения , котрый перенаправит пользователя на страницу с описанием ошибки ,
6. isNotEmptyField - функция ,которая проверяет поле на его содержание.
Если поле пустое то функция вернет False, иначe True
7. filterBooks - функция , которая фильтрует все книги из БД по трем ключам 
8. book_search - функция , которая отображает форму , по запросом которой будет происходить поиск на странице 
При отправке пустой формы на странице будут отображены все книги из БД
9. add_book - функция ,которая отображает страницу с формай для добавления книги в Бд. При успешной отправке формы создается новая книга 
и пользователь перенаправляется на страницу "Отображение всех книг"

forms.py :

1. BooksForm - класс для работы с формай связанной с моделью Book.
Данная форма отображает все поля при создании нового объекта формы 
2. BooksStatusForm - класс для работы с формой связанной с моделью Book.
Данная форма создана для изменения статуса Книги

models.py :

1. Book - модель для создания объекта Book c полями title, author, year, status.
Поле status имеет связь ForeignKey с классом Status
2. Status - модель для создания объекта Status с полем name
